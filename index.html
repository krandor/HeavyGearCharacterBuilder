HeavyGearCharacterBuilder

Character Builder for the Silhouette / Heavy Gear system

Download this project as a .zip file Download this project as a tar.gz file

What is it?

This project is all about creating a tool for the Heavy Gear / Silhouette RPG system. It allows you to fill out the normal character sheet details, save them to HTML local storage for you to keep, and eventually will have the ability to export this information.
